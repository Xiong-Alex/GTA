# Global Travel App

This project now runs as a normal standalone Expo app.

## Run Locally

```powershell
npm install
npx expo start --web
```

You can also use `npx expo start` and open it in Expo Go or a simulator.

## What Works Offline

- Trips use built-in demo data and save locally on the device.
- Notifications, feedback, and FAQs are local.
- Support chat runs in local demo mode for now.

## Later

If you want real APIs or AI later, we can add a backend again. For this prototype, the repo is frontend-only.
